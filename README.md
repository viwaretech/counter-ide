## 📦 Overview

**Counter IDE** is a custom programming tool designed for **ESP8266** and **Arduino** microcontroller devices.

This project aims to provide a **simple, lightweight, and efficient firmware programming environment** for IoT developers and embedded system enthusiasts.

Counter IDE helps simplify the process of uploading firmware and managing connected devices.

---

## ✨ Features

- ⚡ Custom firmware uploader
- 📡 WiFi device support
- 🔧 Arduino compatible
- 📟 OLED display projects
- 💻 Windows based application
- 🚀 Lightweight and easy to use

---

## ✨ installation instructions
Since version 2.1.0 and above, the installer no longer includes SDK packages, you have to download them manually and extract the packages into the installation folder, or you can download and install the older version first (v.2.0.0) then download and install the latest version
